# 4. 统一脚手架基础设置

- 根据不同应用配置基础html模板 & favicon.ico
- 内容加密，接口防爬、图片防盗链
- icon管理 ，和阿里icon同步，都走svg组件，高保真
- axios 封装，保持统一，定制接口写项目里
- 接口状态码、登录机制、接口规范风格统一
- store状态数每个项目独立
- header 、footer等组件定制几套主题，管理端控制渲染
- vue-js-modal 改造所有弹窗
